# Open Source Plan

## Good project targets

1. FastAPI: documentation improvements, small examples, issue reproduction apps, and typing-related fixes.
2. SvelteKit: docs, examples, adapter edge cases, and small reproduction-driven fixes.
3. Playwright: test examples, documentation clarifications, and bug reproductions.
4. Cobra or Charm ecosystem: Go CLI documentation, examples, and small behavior fixes.
5. Supabase or Postgres tooling: docs, starter examples, and integration issue reproductions.

## First issues to look for

- Documentation pages with unclear setup steps
- Broken examples or outdated commands
- Small TypeScript/Python typing improvements
- Tests for already reported bugs
- Reproduction repositories for bugs maintainers cannot reproduce

## How to write issue comments

- Confirm the exact version, OS, and reproduction steps.
- Explain what you expected and what happened.
- Include the smallest reproduction you can.
- Ask whether a PR would be welcome before spending a lot of time.

## How to write PR descriptions

- Start with the problem.
- Explain the change in plain language.
- Link the issue when one exists.
- List how you tested it.
- Mention any tradeoffs or areas where maintainer feedback is needed.

## How to mention external PRs on your profile

Add a small section only after real merged or reviewed work exists:

```md
## Open-source contributions

- Project name: short description of the fix/docs contribution. [PR link]
```

Do not count comments, stars, or forks as contributions. Use only real issues, PRs, reviews, or documentation changes.

## 30-day contribution plan

Week 1: Pick two ecosystems, read contributor guides, reproduce one open issue, and leave one useful issue comment.

Week 2: Submit one small documentation or test PR. Keep the scope narrow and make the maintainer's review easy.

Week 3: Submit one small code PR or example improvement. Include tests or a reproduction.

Week 4: Write a short reflection in your portfolio about what you learned from contributing, linking only real public work.
